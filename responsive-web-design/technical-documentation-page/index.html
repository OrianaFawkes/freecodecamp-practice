Some Outerspace Language

Some Outerspace Language (SOL) is a simple imperative programming language patterned to Java. The language makes use of references about astronomical words such as Big Bang, Orbits, Supernova, etc. In partial fulfillment of the requirements for my Compiler Design course, my group and I made this language along with its compiler written in Java.
